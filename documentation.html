📚 Documentation

Everything You Need to Use UIverse Components

Learn installation, usage, customization, and contribution guidelines for building modern UI faster with UIverse.

⚡ Quick Start

Add UIverse stylesheet:

<link rel="stylesheet" href="style.css">

Use a component:

<button class="primary-btn">Click Me</button>

⚙️ Installation

  • ✔ Clone or download UIverse repository
  • ✔ Include style.css in your project
  • ✔ Start using components directly in HTML

🧩 Component Usage

UIverse provides ready-to-use components like buttons, cards, forms, and navbars. 

<div class="card">
  <h3>Card Title</h3>
  <p>Card description here</p>
</div>

🎨 Customization

You can easily modify colors, spacing, and typography using CSS variables and utility classes. 

:root {
  --primary: #7c3aed;
  --bg: #0f172a;
}

❤️ Contribution Guide

UIverse is open source. You can add new components, improve design, or fix issues.
