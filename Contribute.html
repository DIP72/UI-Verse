Contribute to UIverse 🤝

UIverse is open for everyone. Whether you're a beginner or experienced developer, you can contribute and improve this UI library.

1️⃣ Fork the Repository

Go to the GitHub repository and click on "Fork" to create your own copy.

2️⃣ Clone the Project

Download your forked repository to your local machine using Git.

3️⃣ Add Your Component

Create a new UI component (button, card, form, etc.) using clean HTML, CSS, and JS.

4️⃣ Follow Structure

Keep your code simple, reusable, and properly organized with clear class names.

5️⃣ Commit & Push

Save your changes and push them to your GitHub fork.

6️⃣ Create Pull Request

Submit a Pull Request so your component can be reviewed and added to UIverse.

Contribution Guidelines

✔ Use clean and readable code
✔ Avoid unnecessary libraries
✔ Make components responsive
✔ Add comments where needed
✔ Keep UI modern and simple
